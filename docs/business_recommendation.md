# Business Recommendation

## 1. 目的

レビュー低評価の要因を分析し、事業者が実施可能な改善施策を提案する。

本分析では、EDA（探索的データ分析）、ランダムフォレスト、およびSHAP分析の結果をもとに、低評価レビューとの関連が強い要因を抽出した。

---

## 2. 分析結果サマリ

### EDA結果

EDAでは以下の傾向が確認された。

* 配送遅延が大きいほど低評価率が高い
* 配送日数が長いほど低評価率が高い
* 送料が高い注文ほど低評価率が高い
* 商品カテゴリごとに低評価率の差が存在する
* 地域ごとに低評価率の差が存在する
* 支払方法による差は限定的であった

低評価注文と高評価注文を比較すると、以下の差が確認された。

| 指標       |   高評価 |   低評価 |
| -------- | ----: | ----: |
| 平均配送日数   | 10.9日 | 19.7日 |
| 平均配送遅延日数 | 0.23日 | 3.91日 |
| 平均送料     |  22.0 |  27.7 |
| 平均支払額    | 133.4 | 159.9 |

配送関連指標の差が特に大きく、配送品質がレビュー評価へ強く影響していることが示唆された。

---

### モデル分析結果

ランダムフォレストおよびSHAP分析の結果、低評価レビューに影響する主な特徴量は以下であった。

| 順位 | 特徴量           |
| -- | ------------- |
| 1  | delay_days    |
| 2  | delivery_days |
| 3  | item_count    |
| 4  | total_freight |
| 5  | total_payment |

SHAP分析では、

* 配送遅延が大きいほど低評価リスクが上昇する
* 配送日数が長いほど低評価リスクが上昇する
* 商品数が多い注文ほど低評価リスクが上昇する
* 送料が高い注文ほど低評価リスクが上昇する

ことが確認された。

また、一部の地域およびカテゴリも低評価リスクに寄与していることが確認された。

---

## 3. 提言① 配送品質の可視化と管理強化

### 背景

配送日数および配送遅延日数は、EDA・ランダムフォレスト・SHAPの全てで最も重要な要因として確認された。

州別分析では、低評価率と各指標の相関係数は以下であった。

| 指標       | 低評価率との相関 |
| -------- | -------: |
| 平均配送遅延日数 |    0.395 |
| 平均配送日数   |    0.275 |
| 平均送料     |    0.247 |

配送遅延が最も強い相関を示していた。

### 推奨施策

* 配送遅延が多い地域の監視強化
* 配送業者との定期レビュー実施
* 配送予定日の精度向上
* 配送遅延発生時の顧客通知改善
* 地域別配送品質ダッシュボードの整備

### 留意点

配送品質は低評価レビューの主要因である。

一方で、配送遅延や配送日数は購入後に確定する情報であり、事業者が直接制御できる範囲には限界がある。

そのため配送品質改善のみではなく、事業者が直接改善可能な要因にも注力する必要がある。

---

## 4. 提言② 高送料商品の顧客体験改善

### 背景

送料はEDAおよびSHAP分析の双方で重要な要因として確認された。

低評価注文では平均送料が高評価注文より約26%高かった。

送料が高い注文では、顧客の期待値が高まり、わずかな不満でも低評価につながる可能性がある。

### 推奨施策

* 送料無料ラインの導入
* セット販売による送料負担軽減
* 送料内訳の明確化
* 高送料商品の購入者へのフォロー強化

### 期待効果

顧客の送料に対する不満を軽減し、低評価レビューの削減が期待できる。

---

## 5. 提言③ 多商品注文への対応強化

### 背景

SHAP分析では item_count が上位の重要特徴量として確認された。

商品数が多い注文では、

* 梱包ミス
* 商品不足
* 商品破損
* 配送トラブル

などが発生しやすくなる可能性がある。

配送関連以外で事業者が直接改善できる要因として重要である。

### 推奨施策

* 多商品注文の出荷前チェック強化
* 梱包品質向上
* 欠品管理強化
* 注文分割配送の最適化

### 期待効果

配送業者に依存しない形で低評価レビューの削減が期待できる。

---

## 6. 提言④ 高リスクカテゴリの重点分析

### 背景

カテゴリ別分析では低評価率に大きな差が確認された。

特に以下のカテゴリで低評価率が高かった。

| カテゴリ                    |  低評価率 |
| ----------------------- | ----: |
| fashion_roupa_masculina | 22.9% |
| audio                   | 21.7% |
| moveis_escritorio       | 21.6% |
| casa_conforto           | 17.9% |
| telefonia_fixa          | 16.9% |

またSHAP分析でも、

* cama_mesa_banho
* informatica_acessorios
* telefonia
* relogios_presentes

などのカテゴリが重要特徴量として抽出された。

### 推奨施策

* 商品説明の見直し
* 商品画像の改善
* 初期不良率の確認
* 返品理由の分析
* カテゴリ別レビュー内容の詳細分析

### 期待効果

配送以外の根本原因の特定につながる。

---

## 7. 提言⑤ 地域別改善施策

### 背景

地域別分析では州ごとに低評価率の差が確認された。

特に以下の州では低評価率が高かった。

| 州  |  低評価率 |
| -- | ----: |
| AL | 21.3% |
| MA | 19.9% |
| SE | 18.9% |
| RJ | 18.3% |
| PA | 17.9% |

また低評価率は平均配送遅延日数との正の相関が確認された。

### 推奨施策

* 高リスク地域の重点監視
* 地域別配送品質レポート作成
* 地域別キャンペーンの最適化
* 地域別顧客満足度調査

---

## 8. 結論

本分析では、配送遅延および配送日数が低評価レビューと最も強く関連する要因であることが確認された。

一方で、

* 商品数
* 送料
* 商品カテゴリ
* 配送地域

についても低評価との関連が確認された。

特に商品数および商品カテゴリは、事業者が直接改善可能な要因であり、低評価率改善施策の優先対象として有望である。

今後は、

1. 配送品質の監視・改善
2. 多商品注文の品質管理強化
3. 高送料商品の顧客体験改善
4. 高リスクカテゴリの詳細分析

を優先的に実施することを推奨する。
